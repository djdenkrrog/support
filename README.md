SupportServer:
 Эмулятор работы TAPI сервера. Отправляет на заданный порт номера телефонов входящих звонков.

SupportClient:
 Демо программа для работы службы поддержки. Демонстрация работы с различными технологиями в QT.
 Пользователи для входа:
  1. login:user
     passw:user
  2. login:user1
     passw:user1
 Горячие клавиши:
  Ctrl-S - Сохранить, добавить в базу запись.
  Ctrl-D - Взять в работу
  Ctrl-W - Перенести данные с базы в поле для работы.
