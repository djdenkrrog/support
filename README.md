SupportServer:</br>
 Эмулятор работы TAPI сервера. Отправляет на заданный порт номера телефонов входящих звонков.</br>
SupportClient:</br>
 Демо программа для работы службы поддержки. Демонстрация работы с различными технологиями в QT.</br>
 Пользователи для входа:</br>
  1. login:user</br>
     passw:user</br>
  2. login:user1</br>
     passw:user1</br>
 Горячие клавиши:</br>
  Ctrl-S - Сохранить, добавить в базу запись.</br>
  Ctrl-D - Взять в работу</br>
  Ctrl-W - Перенести данные с базы в поле для работы.</br>
